# Chethana Marasinghe — Portfolio

Modern, animated portfolio built with Next.js (App Router), Tailwind CSS, and framer‑motion-like micro‑interactions. Features a dynamic background, spinning conic ring avatar, typed roles, animated projects, and premium skill badges.

Live: https://your-vercel-domain.vercel.app

## Features

- Next.js App Router + Tailwind CSS
- Dynamic pointer-follow background + subtle aurora blobs
- Spinning conic “square ring” avatar
- Typed headline with matching role icons
- Animated project cards (lift, shine, uniform heights)
- Circular skill badges with rotating ring on hover
- Dark mode toggle (class-based)
- Accessible skip links and semantic structure
- SEO metadata + Open Graph image

## Tech Stack

- Next.js 13+/14+ (App Router)
- Tailwind CSS 3
- React Icons
- Optional: Vercel for hosting
